# TradeMapClone mentoria DIO

APP que simula atualização da bolsa de valores em tempo real para mentoria de Android + Jetpack com Kotlin.

O app segue a arquitetura MVVM e apresenta as seguintes tecnologias:

    MQTT;
    Retrofit2;
    ViewModel;
    LiveData;
    Courotines;
    Koin;
    Navigation;
    Room.
